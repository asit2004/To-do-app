# ✅ MERN Todo App

A full-stack Todo application built using the **MERN** stack — MongoDB, Express.js, React, and Node.js.  
It allows users to create, update, delete, and manage tasks with persistent storage and a clean UI.

---

## 🚀 Features

- ✅ Create, read, update, and delete tasks (CRUD)
- 💾 Persistent storage with MongoDB
- 🔐 RESTful API using Express.js and Node.js
- ⚛️ Responsive frontend built with React
- 🌐 Seamless integration between client and server

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose)
- **Other Tools:** Git, GitHub, Postman, VS Code
